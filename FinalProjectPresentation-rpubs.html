Exploring Auto Fuel Efficiency

Exploring Auto Fuel Efficiency is an interactive web app for simple exploration of the 'mpg' data set in the 'ggplot2' package.

  • The application will:
    • Display the data in a easy to read, sortable table
    • Allow the user to filter the data set to focus on specific manufacturers, transmission types, or number of cylinders
    • Graph either city or highway fuel efficiency against a user selected variable

View and Graph Data

The Exploring Auto Fuel Efficiency app has four main sections

  • Filters: The user can filter the data to show only the selected manufacturer, transmission type, or number of cylinders.
  • Data: The filtered data is displayed in the table in the 'Data' tab at the bottom. A selected row in the data table is highlighted in the graph.
